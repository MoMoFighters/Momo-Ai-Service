# MomoCity 회원 탈퇴 정책

## 1. 탈퇴 시 처리

- 회원 탈퇴를 신청하면 계정은 즉시 **탈퇴 상태**로 전환되어 더 이상 서비스를 이용할 수 없습니다.
- 탈퇴 시점에 사용하던 **닉네임은 삭제**되며, 이후 다른 사용자가 동일한 닉네임을 사용할 수 있게 됩니다.
- 탈퇴 처리 시각이 기록됩니다.

---

## 2. 탈퇴 후 데이터 보관

- 탈퇴한 계정 정보는 즉시 완전히 삭제되지 않고, **탈퇴일로부터 3개월간 보관**된 뒤 완전 삭제(하드 삭제)됩니다.
- 보관 기간이 지나면 계정 정보는 시스템에서 완전히 삭제되어 복구할 수 없습니다.
