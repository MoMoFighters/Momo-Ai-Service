# MomoCity 강사 신청 정책

## 1. 신청 자격

- 강사 신청은 **학생(STUDENT)만** 가능합니다.
- **강의 시청 이력이 없어야** 지원할 수 있습니다.
- 아래 경우에는 신청할 수 없습니다.
  - 이미 강사 신청을 진행 중인 경우 (심사 대기 상태)
  - 이미 강사인 경우

---

## 2. 신청 시 제출 항목

- 닉네임: 기존과 다른 닉네임으로 변경하는 경우, 중복 여부를 확인하며 이미 사용 중인 닉네임이면 신청할 수 없습니다.
- 카테고리(전문 분야)
- 증빙 자료: **필수 제출**이며, 미제출 시 신청이 거부됩니다.

### 증빙 자료 형식 제한
- **pdf 또는 mp4 파일만** 업로드 가능합니다.
- 확장자가 없거나 위 두 형식이 아닌 파일은 업로드할 수 없습니다.

---

## 3. 심사 및 결과 처리

신청 후에는 관리자 심사를 거쳐 **승인** 또는 **반려** 처리됩니다.

### 승인되면
- 강사 권한이 부여되어 정상적으로 강사 활동을 시작할 수 있습니다.
- 신청 시 선택한 카테고리에 맞는 기본 프로필 이미지가 적용됩니다.
- 승인 결과가 이메일로 안내됩니다.

### 반려되면
- 학생 신분이 그대로 유지됩니다.
- 관리자는 **최소 10자 이상**의 반려 사유를 남겨야 하며, 해당 사유가 이메일로 안내됩니다.
- 반려 후에도 다시 강사 신청을 할 수 있습니다.

---

## 4. 신청 포기

- 반려된 이후, 더 이상 강사 신청을 하지 않으려는 경우 "신청 포기" 처리를 통해 일반 학생 상태로 완전히 전환할 수 있습니다.
