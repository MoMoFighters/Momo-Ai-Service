# 멤버십 결제 및 환불 정책

## 1. 플랜 안내

| 플랜 | 월 이용료 |
|---|---|
| BASIC | 0원 |
| PLUS | 29,900원 |
| PRO | 49,900원 |

- 모든 플랜은 **결제일로부터 30일간** 유효합니다.

---

## 2. 결제 시나리오별 정책

### 2-1. 신규 결제 (BASIC → PLUS/PRO)
- 선택한 플랜의 정가를 결제합니다.
- 결제 완료 시점부터 30일간 이용 가능합니다.

### 2-2. 갱신 (같은 플랜 재결제)
- **이용 중인 멤버십 만료일까지 7일 이내**일 때만 갱신할 수 있습니다.
- 갱신 시 결제 금액은 정가 그대로이며, **기존 이용 종료일에 30일이 그대로 이어붙습니다.**
  - 예: 만료 5일 전 갱신 → 새 종료일 = 기존 종료일 + 30일
- 만료까지 7일 넘게 남은 시점에는 갱신 결제를 할 수 없습니다. (이용 종료일이 다가오면 자동으로 갱신 버튼이 활성화됩니다.)

### 2-3. 업그레이드

**BASIC → PLUS / BASIC → PRO**
- 정가를 전액 결제합니다. (잔여일수 계산 없음)

**PLUS → PRO (유료 플랜 간 전환)**
- 결제 금액은 **잔여 이용기간에 정확히 비례한 차액**으로 계산됩니다.

  ```
  결제 금액 = (PRO 가격 − PLUS 가격) × 잔여기간 ÷ 30일
  ```

  - 예: PLUS(29,900원) 이용 중 정확히 15일 남은 시점에 PRO(49,900원)로 업그레이드
    → (49,900 − 29,900) × 15 ÷ 30 = **10,000원**
- 잔여기간은 일 단위로 버리거나 올리지 않고, **남은 시간을 그대로 정확히 반영**하여 계산됩니다. (예: 15일 12시간이 남았다면 그 시간까지 정확히 비례 계산에 반영됩니다.)
- 업그레이드해도 **이용 종료일은 변경되지 않습니다.** 남은 기간 동안 상위 플랜 혜택만 적용됩니다.
- 이용 종료일이 이미 지난 상태에서 업그레이드하는 경우, 신규 결제와 동일하게 정가가 적용됩니다.

### 2-4. 다운그레이드 (PRO→PLUS, PRO→BASIC, PLUS→BASIC)
- 하위 플랜으로 내려가고 싶다면 **구독을 취소**해 주세요. 구독 취소는 환불 처리 결과에 따라 BASIC 또는 남아있는 하위 플랜으로 전환됩니다. (자세한 내용은 3장 참고)

### 2-5. 이미 이용 중인 플랜 재결제
- 같은 플랜을 만료 7일 이전 시점에 다시 결제하려고 하면 "이미 이용 중인 플랜입니다" 안내와 함께 결제가 제한됩니다.

---

## 3. 구독 취소 / 환불 정책

### 3-1. 구독 취소
- 구독 취소는 결제 건에 대한 환불 요청으로 처리됩니다.
- 단일 결제 건만 있는 경우(BASIC에서 바로 PLUS/PRO를 결제한 경우) 환불이 승인되면 **즉시 BASIC 플랜으로 전환**됩니다.
- PLUS에서 PRO로 업그레이드한 이력이 있는 경우, 어떤 결제 건까지 환불되는지에 따라 최종 플랜이 BASIC 또는 PLUS로 달라질 수 있습니다. (자세한 내용은 3-3 참고)

### 3-2. 환불 가능 조건
- **각 결제 건은 결제 완료 후 3일 이내**에만 환불 요청이 가능합니다.
- 3일이 지난 결제 건은 환불되지 않으며, 해당 결제로 얻은 혜택은 이용 종료일까지 그대로 유지된 뒤 자동으로 BASIC 플랜으로 전환됩니다.
- 아래의 경우 환불 요청이 거부됩니다.
  - 본인이 결제한 건이 아닌 경우
  - 이미 환불 처리가 완료된 건인 경우
  - 이전에 환불 시도가 있었으나 처리에 실패해 수동 확인이 필요한 상태인 경우

### 3-3. PLUS → PRO로 업그레이드한 경우의 환불
같은 구독 기간 안에서 BASIC → PLUS 결제 후, 이어서 PLUS → PRO로 업그레이드한 경우, PRO 결제 건에 대해 환불을 요청하면 아래와 같이 처리됩니다.

- **두 결제 건이 모두 3일 이내인 경우**: PRO 결제와 PLUS 결제가 **함께 환불**되며, 플랜은 즉시 BASIC으로 전환됩니다.
- **PRO 결제만 3일 이내이고 PLUS 결제는 3일이 지난 경우**: PRO 결제만 환불되며, 플랜은 BASIC이 아닌 **원래의 PLUS로 전환**됩니다. 이때 PLUS 이용 종료일은 처음 PLUS를 결제했던 시점 기준으로 그대로 유지됩니다.
- **두 결제 건 모두 3일이 지난 경우**: 환불이 불가능하며, 이용 종료일까지 현재 플랜(PRO)을 그대로 이용하신 뒤 자동으로 BASIC 플랜으로 전환됩니다.

반대로 PLUS 결제 건만 단독으로 환불을 요청하는 경우, **이미 PRO로 업그레이드되어 PRO 결제가 아직 유효하게 남아있다면 환불이 제한**됩니다. 이 경우 PRO 결제 건부터 먼저 환불을 진행해 주세요.

### 3-4. 환불 처리
- 환불 승인 시 결제 대행사를 통해 실제 결제 취소가 진행되며, 처리와 동시에 해당하는 플랜으로 전환됩니다.
- 여러 결제 건을 함께 환불하는 경우, 일부 건의 결제 취소 처리가 실패할 수 있습니다. 이 경우 정상 처리된 건은 그대로 환불 완료되고 플랜에 반영되며, 실패한 건만 별도 확인 절차(수동 처리)를 거칩니다. 이 경우 해당 건의 환불이 지연될 수 있습니다.

## 4. 결제 진행 중 발생할 수 있는 안내

| 상황 | 안내 |
|---|---|
| 이미 진행 중인 결제가 있는 경우 | 새 결제 요청을 할 수 없습니다. 기존 결제를 완료하거나 취소 후 다시 시도해 주세요. |
| 결제 금액이 실제 승인 금액과 다른 경우 | 결제가 자동으로 취소 처리되며, 다시 결제를 진행해 주세요. |
| 결제가 정상적으로 완료되지 않은 경우 | 결제 실패로 처리되며, 다시 결제를 진행해 주세요. |
| 이미 처리 완료된 결제 건을 다시 조회한 경우 | 기존 처리 결과가 그대로 안내됩니다. |
| 이미 환불된(또는 환불 시도 이력이 있는) 결제 건에 다시 환불을 요청한 경우 | 환불이 제한됩니다. |
