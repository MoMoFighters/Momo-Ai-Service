# 포인트 적립 및 사용 안내

## 포인트 적립 방법
아래 활동을 하면 포인트가 적립됩니다.

- **강의 챕터 완료**: 챕터 하나를 완료할 때마다 10포인트 적립
- **방명록 작성**: 하루 1회에 한해 방명록을 작성하면 10포인트 적립 (하루에 여러 번 작성해도 1회분만 적립)
- **수강평 작성**: 하나의 강의당 수강평은 한 번만 작성할 수 있으며, 작성 시 30포인트 적립 (같은 강의에 중복으로 수강평을 작성할 수는 없습니다)
- **친구 관계 맺기**: 새로운 친구가 생길 때마다 3포인트 적립

적립된 포인트는 보유 포인트가 사용하려는 금액 이상일 때 강의 구매 등에 사용할 수 있습니다.

## 포인트 사용처
아래 활동을 이용하면 포인트가 차감됩니다.

- **친구 도시 방문(버스비)**: 1포인트 차감
- **운세 보기**: 5포인트 차감
- **상점 아이템 구매**: 아이템 가격만큼 차감 (가격은 상점에서 직접 확인)
