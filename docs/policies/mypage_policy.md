# MomoCity 마이페이지 / 내 정보 정책

## 1. 마이페이지에서 조회 가능한 정보

- 프로필 이미지
- 이메일
- 이름
- 닉네임
- 보유 포인트
- 알림(밤 시간대 알림) 설정 여부
- 멤버십 플랜 및 시작일 / 만료 예정일
- 임시 비밀번호 사용 여부
- 가입일
- 보유 건물 목록

---

## 2. 내 정보 수정

### 프로필 이미지
- 보유하고 있는 프로필 아이템 중에서 선택하여 변경할 수 있습니다.

### 닉네임
- 원하는 닉네임으로 변경할 수 있습니다.
- **이미 다른 사용자가 사용 중인 닉네임**으로는 변경할 수 없습니다.

### 비밀번호
- 비밀번호를 변경하려면 **현재 비밀번호를 반드시 입력**해야 합니다.
- 입력한 현재 비밀번호가 실제 비밀번호와 **일치해야** 변경이 진행됩니다.
- **기존 비밀번호와 동일한 비밀번호로는 변경할 수 없습니다.**

---

## 3. 알림 설정

- 알림을 켜고 끌 수 있으며, 밤 시간대에 한정되지 않고 언제든 설정할 수 있습니다. 다시 요청하면 이전 설정과 반대로 전환됩니다.
