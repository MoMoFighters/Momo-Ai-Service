# 커뮤니티 게시글 및 댓글 규칙

## 작성/수정/삭제 권한
게시글과 댓글은 **작성한 본인만** 수정하거나 삭제할 수 있습니다. 다른 사람이 작성한 게시글이나 댓글은 수정/삭제할 수 없습니다.

## 게시글 작성 규칙
- 게시글에 이미지를 첨부하는 경우 **최대 5장까지** 업로드할 수 있습니다.
- 텍스트 게시글은 본문 내용이 필요하고, 이미지 게시글은 이미지 URL이 필요합니다.
- 이미지를 업로드한 경우에는 반드시 썸네일을 지정해야 합니다.

## 댓글/대댓글 규칙
- 삭제된 게시글에는 댓글을 작성할 수 없습니다.
- 삭제된 댓글에는 답글(대댓글)을 달 수 없고, 삭제된 게시글의 댓글에도 답글을 달 수 없습니다.
- **답글은 한 단계까지만 가능합니다.** 즉, 대댓글(답글)에는 다시 답글을 달 수 없습니다.
- 답글이 달리면, 원 게시글 작성자와 답글이 달린 댓글의 작성자에게 알림이 갑니다 (답글 작성자 본인은 제외).

## 삭제된 게시글/댓글 보관
게시글이나 댓글을 삭제해도 즉시 완전히 사라지지 않고, 일정 기간 보관 후 완전히 삭제됩니다. 삭제 처리된 시점으로부터 **6개월이 지나면** 완전히 삭제되어 복구할 수 없습니다.
