# MomoCity 회원가입/로그인 정책

## 1. 회원가입

- 회원가입 전 **이메일 인증**이 필요합니다.
  - 인증코드는 **6자리 숫자**이며, 발송 후 **3분간만 유효**하며, 시간이 지나면 재발송해야 합니다.
- **이미 가입된 이메일**로는 다시 가입할 수 없습니다.
- 가입 시 기본 권한은 **학생(STUDENT)**으로 부여됩니다.
- **최초 로그인 시 닉네임을 등록**해야 합니다. 닉네임은 모모시티 내에서 사용되는 활동명이며, 이후 마이페이지 등에서 **자유롭게 수정**할 수 있습니다.

---

## 2. 로그인

### 로그인 방식
- 가입한 이메일과 비밀번호로 로그인합니다.

### 계정 상태에 따른 로그인 제한
계정 상태에 따라 로그인이 제한될 수 있습니다.

| 상태 | 로그인 가능 여부 | 안내 문구 |
|---|---|---|
| 강사 승인 대기중 | 불가 | "강사 승인 대기중입니다." |
| 정지 (1~2회 신고 누적) | 불가 | "정지된 계정입니다. 문의 사항이 있다면 yourmomocity@gmail.com 으로 문의 주시길 바랍니다." |
| 영구 정지 (3회 이상 신고 누적) | 불가 | "정지 3회 누적으로 인해 계정이 영구적으로 정지 되었습니다." |
| 그 외 로그인 불가 상태 | 불가 | "해당 계정은 현재 로그인이 불가능한 상태입니다." |
| 정상(활성) / 강사 반려 상태 | 가능 | - |

---

## 3. 비밀번호를 잊은 경우 (임시 비밀번호)

- **가입된 이메일에 한해서만** 임시 비밀번호를 발급받을 수 있습니다.
- 발급된 임시 비밀번호는 이메일로 전송됩니다.
- 임시 비밀번호로 로그인한 경우, 로그인 세션이 **3분으로 제한**되며, 그 안에 반드시 새 비밀번호로 변경해야 합니다. 시간이 지나면 다시 임시 비밀번호를 발급받아야 합니다.

---

## 4. 소셜 로그인 (카카오 / 구글 / 네이버)

- 소셜 로그인은 최초 이용 시 자동으로 회원가입 처리되며, 기본 권한은 학생(STUDENT)입니다.
- **이미 이메일/비밀번호로 가입된 이메일**은 동일 이메일의 소셜 계정으로 새로 가입할 수 없습니다. 이 경우 기존 방식(이메일/비밀번호)으로 로그인하거나 다른 이메일을 사용해야 합니다.

---

## 5. 로그아웃

- 로그아웃하면 해당 로그인 세션은 즉시 만료되어, 그 이후로는 재사용할 수 없습니다.

---

## 6. 로그인 세션 연장

세션(로그인 상태)을 연장하려는 경우에도 아래 계정은 연장이 제한됩니다.

- 강사 승인 대기중인 계정
- 임시 비밀번호를 아직 변경하지 않은 계정 (새 비밀번호로 변경해야 연장 가능)
